# 特定商取引法に基づく表記（テンプレート）

> 有償提供（サブスク・買い切り等）を行う場合、特定商取引法によりこの表記の掲示が必要です。
> 以下の `[要記入]` を**実際の事業者情報**で埋め、サイト内に常時アクセスできるページとして掲示してください
> （例：`app/tokushoho/page.tsx` を作成し、フッターからリンク）。記載内容は最終的に専門家（弁護士等）の確認を推奨します。

| 項目 | 記載内容 |
| --- | --- |
| 販売事業者 | [要記入：個人事業主は氏名、法人は法人名] |
| 運営統括責任者 | [要記入：氏名] |
| 所在地 | [要記入：住所。請求があれば遅滞なく開示する旨の記載も可] |
| 連絡先（電話） | [要記入：電話番号。請求があれば遅滞なく開示でも可] |
| 連絡先（メール） | [要記入：問い合わせ用メールアドレス] |
| 販売価格 | [要記入：税込価格／プラン別価格] |
| 商品代金以外の必要料金 | [要記入：通信料は利用者負担、等] |
| 支払方法 | [要記入：クレジットカード等] |
| 支払時期 | [要記入：申込時／毎月◯日、等] |
| サービス提供時期 | [要記入：決済完了後ただちに利用可能、等] |
| 返品・キャンセル（返金ポリシー） | [要記入：デジタルサービスのため原則返金不可／中途解約の扱い等] |
| 動作環境 | 最新版の主要ブラウザ（Chrome / Safari / Edge）。JavaScript有効。 |
| 事業者の特別な販売条件 | [任意] |

## 掲示の実装メモ
- `app/tokushoho/page.tsx` を `/terms`・`/privacy` と同じ体裁（`legal-page`）で作成。
- `app/page.tsx` フッターの `.footer-links` に `<a href="/tokushoho">特定商取引法に基づく表記</a>` を追加。
- 無料提供のうちは掲示不要だが、課金導線を出す前に必ず用意すること。
